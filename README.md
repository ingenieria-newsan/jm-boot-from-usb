## Modo operativo

2. Crear carpeta ``S:\EFI\JMBoot``.
3. Copiar archivo ``uefi_shell_min.efi`` como ``S:\EFI\JMBoot\bootx64.efi``.

6. Modificar el nombre de la entrada a ``System Image Deploy``.
7. Establecer la entrada ``{bootmgr}`` como ``default``.
